# eCommerce Platform

## Descripción
Este proyecto es una plataforma de comercio electrónico diseñada para la venta de dispositivos. Incluye una variedad de páginas y componentes que permiten a los usuarios navegar, buscar y comprar productos de manera eficiente.

## Estructura del Proyecto
```
ecommerce-platform
├── src
│   ├── assets
│   │   └── styles
│   │       └── main.css        # Estilos CSS para la plataforma
│   ├── components
│   │   ├── Header.html         # Componente de encabezado
│   │   ├── Footer.html         # Componente de pie de página
│   │   └── ProductCard.html    # Componente para mostrar tarjetas de productos
│   ├── pages
│   │   ├── index.html          # Página principal
│   │   ├── products.html       # Página de productos
│   │   ├── offers.html         # Página de ofertas
│   │   └── contact.html        # Página de contacto
│   └── scripts
│       └── main.js             # Scripts JavaScript para la funcionalidad
├── package.json                 # Configuración de npm
└── README.md                    # Documentación del proyecto
```

## Instalación
1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias.

## Uso
- Abre `src/pages/index.html` en tu navegador para ver la página de inicio.
- Navega a las diferentes páginas utilizando el menú de navegación.

## Características
- Diseño responsivo que se adapta a diferentes dispositivos.
- Componentes reutilizables para una mejor organización del código.
- Funcionalidad de JavaScript para interacciones dinámicas.
